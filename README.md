# IIT Goa Sports Website

This project is a dynamic sports website for IIT Goa, built using **HTML**, **CSS**, and **JavaScript**. The website serves as a hub for athletic excellence and community spirit, providing users with easy access to the latest schedules, announcements, and information about the sports facilities at IIT Goa.

You can visit the live version of the website here: [IIT Goa Sports Website](https://cs-102.github.io/CS102-group-project/)

---

## Features of the Website

### 1. **Daily Schedule**
The website displays the daily schedule of all sports available at IIT Goa, which can be accessed easily.

### 2. **Announcements**
Coaches can log in to make announcements or delete them, keeping users updated with the latest information.

### 3. **Login and Sign-up**
Coaches have the ability to log in or sign up for accounts. This allows them to make new announcements and manage their schedules.

---

## Pages and Structure

### 1. **Home Page**
The home page is designed to welcome users to the IIT Goa Sports community. It includes:
- Information about each sport available at the institute.
- A dedicated "About" section explaining the purpose and features of the website.
- A responsive navigation bar (navbar) for easy access to different sections.
- A footer with additional information and useful links.

### 2. **Information Page**
The information page provides:
- Details about the daily sports schedule.
- Announcements made by the coaches.
- A user-friendly interface with a consistent navbar and footer.

### 3. **Login and Sign-up Page**
The login and sign-up pages are designed for coaches to:
- Create new accounts, which are stored locally.
- Log in to make new announcements or manage existing ones.

### 4. **Announcements Page**
Once logged in, coaches can:
- Make new announcements using pre-made templates.
- Delete old announcements when they are no longer relevant.

---

## Technologies Used

The following technologies were used to build this website:

- **HTML**: Structure of the web pages.
- **CSS**: Styling and layout for a responsive and visually appealing interface.
- **JavaScript**: Dynamic interactions and functionality, including handling user input and managing announcements.
